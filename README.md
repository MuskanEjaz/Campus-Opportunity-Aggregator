# Campus-Opportunity-Aggregator
Campus Opportunity Aggregator is a web-based platform that centralizes student opportunities such as internships, scholarships, workshops, and competitions into one organized system. It helps students easily discover relevant opportunities that are otherwise scattered across different sources.

The platform provides search and filtering features based on categories, departments, and deadlines, along with options to bookmark opportunities and receive personalized recommendations. A responsive design ensures smooth usage across all devices.

An admin panel allows authorized users to manage listings by adding, updating, or removing opportunities, keeping the platform accurate and up to date.

Built using modern technologies like React, Tailwind CSS, Node.js, and MySQL, the project demonstrates practical skills in frontend development, backend integration, and database management.
